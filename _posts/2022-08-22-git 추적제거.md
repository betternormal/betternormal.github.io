- 이미 git으로 추적되고 있는 파일, 폴더를 추적하지 않게 변경하는 방법

## 단일파일
```
// 해당파일 .gitignore에 추가

// 커맨드 실행
$ git rm --cached <filename>

// 커밋메세지 작성
$ git commit -m "<Message>"

```

## 폴더
```
// 해당폴더 .gitignore에 추가

// 커맨드 실행
git rm --cached -r <folder>

// 커밋메세지 작성
git commit -m "<Message>"
```
