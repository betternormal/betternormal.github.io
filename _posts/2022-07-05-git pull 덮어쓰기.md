로컬 브랜치가 원격과 동기화가 되지 않을 때, 원격의 내용으로 강제로 덮어쓰기

## Master branch
```
$ git fetch --all
$ git reset --hard origin/master
```

## Other branches
```
$ git fetch --all
$ git reset --hard origin/<BRANCH-NAME>
```
