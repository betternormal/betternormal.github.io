### 주석
```dart
// comment

/*
  multi line comment
*/

/// document comment
메서드나 클래스 정의 위에 작성하면 dartdoc과 같은 문서생성도구로 문서가 자동생성된다
```
### 변수
```dart
int num = 10;
double num2 = 20.0;
String name = 'benKim';
bool isTrue = true;

// dart는 타입추론을 지원하기 때문에 var로 선언후 사용가능
var i = 10;
var s = 'hello';
```

### 상수
> 변하지 않아야 하는 값은 상수로 선언후 사용된다. final키워드가 사용된다.
```dart
final name = 'benKim';
name = 'leeKim'; // Error
```
